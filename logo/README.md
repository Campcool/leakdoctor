# 灰汰郎下載用壓縮圖檔

這裡是可直接下載、分別套用的壓縮圖檔。

## 建議使用方式

- 網站顯示：優先使用 `.webp`，檔案最小、速度較好。
- 社群平台上傳：使用 `.jpg` 或 `.png`。
- 透明背景 logo：使用 `.png`；若網站支援 WebP，也可用 `.webp`。
- favicon / app icon：使用 `avatars-icons` 資料夾內的尺寸。

## 資料夾

- `logos/`：網站 logo、深色底反白版、email 簽名檔
- `social-ads/`：Google 廣告、OG、FB、IG、限動尺寸
- `avatars-icons/`：FB/IG/LINE 頭像、favicon、Apple/Android icon

## 網站常用優先檔

- Header logo：`logos/website-header-logo-640x240.webp`
- Header logo fallback：`logos/website-header-logo-640x240.png`
- OG 分享圖：`social-ads/og-facebook-linkedin-1200x630.webp`
- OG fallback：`social-ads/og-facebook-linkedin-1200x630.jpg`
- 深色 OG：`social-ads/og-navy-1200x630.webp`
- 網站 favicon：`avatars-icons/favicon.ico`
- Apple touch icon：`avatars-icons/apple-touch-icon-180.png`

## 廣告常用

- Google 1.91:1：`social-ads/google-display-1_91x1-1200x628.webp`
- Google 1.92:1：`social-ads/google-display-1_92x1-1200x625.webp`
- Google 方形：`social-ads/google-display-square-1200x1200.webp`
- Google 4:5：`social-ads/google-display-4x5-1200x1500.webp`
- Landscape logo：`social-ads/google-ads-landscape-logo-1200x300.webp`

## 社群常用

- FB/IG 頭像白底：`avatars-icons/avatar-white-1024.webp`
- FB/IG 頭像深藍底：`avatars-icons/avatar-navy-1024.webp`
- IG/FB 方形貼文：`social-ads/instagram-facebook-post-1x1-1080.webp`
- IG 4:5：`social-ads/instagram-portrait-4x5-1080x1350.webp`
- 限動/Reels：`social-ads/story-reels-9x16-1080x1920.webp`

## 檔案清單
