看症狀找原因

你看到什麼？

點選你家的漏水症狀，查看可能原因、建議工法與參考費用

💧
天花板持續滴水或大面積水漬
緊急處理管路問題
▼
可能原因樓上給水管破裂，或浴室地板防水層失效。若持續滲水（不分晴雨）幾乎確定是給水管問題。
建議工法壓力測試定位破裂點 → 換管或重做防水層
參考費用$3,000–8,000（依換管難度）
注意事項先關掉總水閥觀察是否停止滴水，可初步判斷是給水管還是排水管問題。
🌧️
下雨後天花板或牆面才出現水漬
盡快處理防水層問題
▼
可能原因頂樓防水層老化、外牆裂縫或窗框嵌縫開裂，讓雨水滲入。
建議工法熱像儀定位滲水路徑 → 防水塗料或重做防水層
參考費用$6,000–20,000（依面積與工法）
注意事項乾燥期間熱像儀效果有限，建議雨後 24 小時內進行檢測效果最佳。
🪟
窗框四周雨天滲水
注意觀察打針可解
▼
可能原因窗框與牆體之間的嵌縫材料老化開裂，或窗框本身變形，雨水從縫隙滲入。
建議工法高壓灌注打針 2–5 針，外側重新嵌縫
參考費用$1,500–4,000（視裂縫數量）
注意事項窗框漏水是相對容易處理的案例，成功率高，適合作為平台初期承接的標準案件。
🟤
牆面白色粉狀結晶（壁癌）
長期問題需找水源
▼
可能原因水泥牆長期受水侵蝕，鈣離子析出形成碳酸鈣結晶。是漏水的結果，不是原因。需先找到滲水源。
建議工法找到外部水源 → 矽酸質防水材處理
參考費用$2,500–5,000/坪（不含外部防水）
注意事項⚠️ 只做表面壁癌處理而不找水源，100% 會復發。必須從滲水側阻水才能根治。
🚿
浴室地板使用後持續潮濕、樓下滲漏
盡快處理需敲磚
▼
可能原因浴室地磚下防水層破損，淋浴水滲入樓板，導致樓下天花板滲漏。
建議工法色素積水測試定位 → 局部或全面敲磚重做防水層
參考費用$15,000–40,000（依浴室大小）
注意事項需要與樓下住戶協調，建議先確認漏水位置再決定敲磚範圍，避免過度施工。
💸
水費異常增加 / 水錶不停轉
緊急暗管破裂
▼
可能原因給水暗管破裂，水持續流失。關閉所有水龍頭後水錶仍轉動，幾乎確定是暗管問題。
建議工法管路定壓測試 + 聲音探測定位 → 打牆換管
參考費用$5,000–15,000（依定位難度）
注意事項立即關閉家中總水閥，避免持續漏水浸濕結構，再聯繫師傅處理。
❄️
冷氣附近天花板潮濕或滴水
一般處理排水管問題
▼
可能原因冷氣排水管堵塞或破裂，導致排水逆流溢出。夏天使用冷氣期間最常發生。
建議工法內視鏡探查排水管 → 疏通或更換部分管路
參考費用$2,000–6,000
注意事項先確認是冷氣排水問題還是結構漏水，兩者處理方式完全不同。先關冷氣觀察是否停止。
⚠️ 重要提醒：壁癌、純潮濕、結露反潮等情況，因含水量不足，部分色素測試無法施作。需先判斷是否為「活性滲水」，再決定工法。請傳照片讓我們先評估，不收費用。
找到漏水點

檢測工法說明

找不到漏水點就貿然施工，是浪費錢的最常見原因。先用對工具定位，再施工才有效。

🌡️ 紅外線熱像儀
$8,000–20,000
非破壞性大面積掃描
透過溫度差異辨識牆體內隱藏的含水區域。可快速掃描大面積，找出肉眼看不到的滲水路徑。是複雜案件的首選工具。
✓ 優點
非破壞・快速・範圍大
✗ 限制
乾燥期溫差小效果有限
🔬 高解析內視鏡
$6,000–12,000
管路專用視覺確認
將鏡頭深入排水管、冷氣排水管或管道間，直接拍攝管內狀況。精確定位破裂點，避免盲目打牆找位置。
✓ 優點
直視管內・定位精確
✗ 限制
受管徑與彎曲度限制
🎨 色素積水測試
含於測試費
追蹤路徑三色辨識
使用紅、藍、綠三色色素注入疑似漏水位置，追蹤滲水路徑。頂樓九宮格分割測試可系統性找出漏水點。
✓ 優點
路徑清晰・多點同測
✗ 限制
需要有水才能流動
💧 管路加壓測試
$3,000–6,000
給水管專用
對給水管路加壓，觀察壓力是否下降，確認管路有無破損。定壓測試可判斷漏水速率，是暗管破裂的基本確認方法。
✓ 優點
客觀數據・費用低
✗ 限制
無法定位破裂點位置
🔊 漏水聲音探測
含於測試費
隱蔽管路
聲波探測儀聆聽水流異音，搭配管路路徑探測器找出埋藏在水泥牆內的給水管破裂位置。
✓ 優點
不打牆即可初步定位
✗ 限制
環境噪音影響準確度
💦 日本水分計
含於勘查費
輔助判斷
高精度水分計測量牆面含水量，區分活性漏水與乾燥舊漬。避免對已無水的舊漬做無謂測試，節省費用。
✓ 優點
快速篩查・低成本
✗ 限制
僅輔助判斷非定位
💡 多種儀器組合使用效果最好。專業師傅通常會依現場狀況選用 2–3 種工具交叉驗證，避免誤判。若您的漏水超過 3 個月找不到原因，建議直接採用儀器檢測，勘查費最低 $3,000，施工超過 $30,000 可折抵。
修好漏水

施工工法說明

不同漏水原因對應不同工法。選錯了花錢也沒用，先看懂再跟師傅討論。

💉 高壓灌注（打針）
$500–1,000 / 針
治標快速小範圍
以環氧樹脂或親水性發泡樹脂，高壓注入裂縫填補止漏。適合窗框、牆角等小範圍裂縫。效果約 1–3 年，裂縫日後可能再開。
✓ 優點
當日施工・費用低廉
✗ 缺點
治標・效果期限短
🎨 防水塗料施作
$600–1,200 / 坪
防水層修復大面積
塗佈 PU 防水膜、彈泥或矽酸質防水材，用於頂樓、外牆、浴室等大面積防水層修復。矽酸質材可滲入牆體形成晶體持久阻水。
✓ 優點
大面積有效・保固較長
✗ 缺點
需等完全乾燥才能驗收
🚰 管路更換
$3,000–8,000 / 次
治本可能需打牆
切開水泥牆或地板，更換老化破裂的給排水管。新式大樓排水管懸吊於下層天花板，可直接替換不需打牆，老屋則多需開鑿。
✓ 優點
根治・一勞永逸
✗ 缺點
工期長・費用相對高
🔨 浴室防水層重做
$3,000–8,000 / 坪
治本屋齡 15 年以上建議
敲除地磚或牆磚，重做防水層後復原。徹底解決浴室漏水的方式。工期約 5–10 天，期間無法使用浴室。
✓ 優點
最根本的解決方式
✗ 缺點
費時・需臨時替代浴室
🧱 矽酸質壁癌處理
$2,500–5,000 / 坪
阻水結晶需找水源
磨除壁癌面，嚴重時打除至紅磚面。以矽酸質防水材分兩次塗刷，滲入牆體形成樹枝狀結晶填補孔隙，從內部阻水。
✓ 優點
從牆體內部阻水
✗ 缺點
未找水源仍會復發
🏠 頂樓防水層重做
依面積估算
整合工法頂樓專用
移除舊隔熱磚，重做防水層後鋪設新隔熱磚。九宮格積水測試確認排水均勻後施工，同時解決積水與隔熱問題。
✓ 優點
防水 + 隔熱一次解決
✗ 缺點
費用較高・工期較長

🔍 初步判斷：冷氣排水管問題
冷氣附近天花板或牆面潮濕，多數是排水管堵塞或破裂導致逆流。

建議做法：先關閉冷氣，確認潮濕是否停止。若停止，確認是排水管問題，可用內視鏡探查後疏通或換管。

參考費用：$2,000–6,000
緊急程度：一般，但長期不處理會損壞天花板材料。

🔍 初步判斷：頂樓防水層老化
頂層住戶下雨天花板滲水，幾乎是頂樓防水層老化失效。屋齡超過 15 年的頂樓防水層需要重新施作。
