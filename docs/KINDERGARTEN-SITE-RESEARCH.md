# 臺灣主要幼兒園官網研究（含安親班）— 新站規劃參考

> 研究日期：2026-07-18（Asia/Taipei）。目的：業主準備製作幼兒園（含安親班）官網，先研究臺灣五大知名幼兒園品牌官網的功能與設計重點，作為資訊架構與轉換設計的依據。
> 研究限制：本次雲端環境網路政策封鎖外部網站直接存取（curl／WebFetch 對目標網域皆 403），資料來源為搜尋引擎結果與各站被索引的頁面摘要，未能逐頁瀏覽截圖。後續若在無網路限制環境，建議補一輪真機瀏覽驗證。

- **康橋**：招生專區（入學年齡、招生流程、說明會、常見問題）、線上招生登記平台、各校區分站、課程特色（游泳、康橋 Time、探究實作、社團）。

## 2. 主要功能歸納（出現頻率由高到低）

1. **課程介紹（分齡呈現）**：五家全有。慣例是按年齡段切（0-3／3-6／國小課後），而不是按科目切；每段講理念＋一天作息＋學習領域。
2. **招生資訊＋轉換入口**：康橋做成「招生專區」（流程、年齡對照、FAQ、說明會登記）；吉的堡做「預約參觀」＋免費體驗課（價值 $500）＋到訪禮（禮券）等誘因。**預約參觀是幼兒園官網的第一轉換目標**（不是線上報名成交）。
3. **分校／據點查詢**：連鎖品牌全有，吉的堡支援縣市＋行政區兩層篩選，各分園頁帶電話與地址。單園官網則以交通位置＋Google 地圖取代。
4. **師資團隊**：老師照片、資歷、證照。家長信任關鍵。
5. **校園環境寫真／影片**：教室、遊戲區、戶外空間、安全設施實景。
6. **最新消息／活動**：招生季（春季班、秋季班）、參觀日、節慶活動。
7. **親子／親職內容專欄**：吉的堡「親職專欄」寫「安親班怎麼選」這類搜尋型文章做 SEO 集客——與灰汰郎 articles/ 的知識文策略相同。
8. **家長專區（登入型服務）**：電子聯絡簿、平安到班通知、繳費紀錄、課表（何嘉仁 HESS Kids；市面另有小安同學等 App）。大型品牌用 App，官網只放入口連結；新站第一階段可先不做，用 LINE 官方帳號補位。
9. **多語切換**：何嘉仁、快樂瑪麗安有中英版（雙語定位的品牌訊號）。
10. **加盟專區**：連鎖品牌才需要，單園不用。
11. **社群連結**：FB／IG／YouTube／LINE 全員標配。

## 3. 設計重點觀察

- **色彩與視覺**：明亮柔和的暖色系＋童趣插畫為主流，但**信任來自真實照片**——園所實景、孩子活動照（需家長授權＋去識別）比卡通素材更有說服力。插畫當裝飾，照片當證據。
- **首頁編排慣例**：Hero（品牌主張＋孩子笑臉）→ 分齡課程卡 → 特色／理念 → 環境與師資 → 活動消息 → 預約參觀 CTA。
- **信任元素**：立案字號與證照、評鑑績優標示（如三之三嘉新幼兒園直接把「評鑑績優」放進標題）、師資證照、安全設施說明、家長見證。
- **表單精簡**：報名／預約表單 6-8 欄位（家長姓名、電話、孩子年齡、意向課程），送出後明確告知回覆時程；提供 LINE／電話多管道。
- **手機優先**：家長主要透過手機＋LINE 溝通，預約流程要能在 LINE 內建瀏覽器完成（灰汰郎已踩過的坑直接適用）。
- **SEO 打法**：地區關鍵字（「○○區 幼兒園」「○○ 安親班」）＋育兒知識文＋Google 商家檔案。第三方評價站（如 preschool.rating.tw 育兒趣，收錄全台 6,952 所幼兒園）與教育部「全國教保資訊網」是家長比價入口，官網資訊要與之一致。

## 4. 幼兒園＋安親班新站的規劃建議

### 4.1 資訊架構（單園版，非連鎖）

```
首頁
├── 關於我們（理念、園長的話、立案資訊、師資團隊）
├── 幼兒園課程（分齡：幼幼班／小班／中班／大班；一日作息；學習領域）
├── 安親班／課後照顧（課輔內容、才藝、寒暑假班、接送與供餐）
├── 校園環境（實景照片／影片、安全設施、教室與戶外空間）
├── 招生資訊（年齡對照、流程、收費方式、補助說明、FAQ、預約參觀）
├── 活動與消息（招生公告、節慶活動、成果展）
├── 親子專欄（SEO 知識文：如何選幼兒園／安親班、補助懶人包…）
└── 聯絡我們（地圖、電話、LINE、預約表單）
```

### 4.2 轉換設計（可直接沿用灰汰郎既有能力）

- 主 CTA＝**預約參觀**：全站固定入口（header＋浮動鈕＋頁尾），開共用 modal 表單 → 組訊息 → LINE 官方帳號預填——與 `header.js` 的 `ldOpenQuote()` 模式完全同構，可抽出改造復用。
- GA4 事件同套路：`visit_open`／`visit_submit`／`line_click`。
- 誘因設計參考吉的堡：免費體驗課＋到訪小禮，降低首訪門檻。

### 4.3 法規紅線（文案與定位必查，比清潔業嚴格得多）

- 幼兒園受《幼兒教育及照顧法》管制：**不得以全外語／分科外語教學招生宣傳**（「全美語幼兒園」是快樂瑪麗安等以補習班立案經營的模式，幼兒園立案者不可這樣寫）；招生廣告不得誇大不實，主管機關會查核（新聞曾點名何嘉仁、吉的堡分園違規）。
- 「安親班」一詞的合法載體是**兒童課後照顧服務中心**（依兒少法）或短期補習班附設課輔（依補習班準則），兩者立案依據不同，網站上的名稱、立案字號、收費揭露要與實際立案一致。
- 兒童照片需取得家長書面授權；未授權者去識別（灰汰郎案例照 SOP §3 可直接沿用）。
- 收費項目與退費辦法各縣市有規範，建議招生頁明確揭露。

### 4.4 技術建議

- 純靜態站（同 leakdoctor 架構：GitHub Pages＋共用 header.js＋JSON-LD）即可滿足第一階段；電子聯絡簿等登入型功能不要自建，先用 LINE 官方帳號群發＋既有市售 App。
- Schema：`Preschool`（LocalBusiness 子類型）＋ `FAQPage` ＋ `Article`；地區頁打「行政區＋幼兒園／安親班」關鍵字。
- 建議另開獨立 repo 與網域，不與灰汰郎混用（品牌、LINE 帳號、GA4 皆需分開——注意已知陷阱 §6-3：LINE 連結絕不可跨品牌混用）。
