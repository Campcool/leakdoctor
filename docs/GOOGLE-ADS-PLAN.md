## 2. 目標與 KPI

- **主要轉換**：`quote_submit`（表單送出）＝一個有姓名電話地址的準客戶。
- **次要（微轉換）**：`line_click`（加好友/開 LINE）、`quote_open`（打開表單）。
- **建議 KPI（首月為學習期，數字先抓感覺、有數據再校準）**：
  - 目標每次轉換成本（CPA）：**漏水**客單高（數千～上萬），可容忍較高 CPA（例如 NT$400–800/表單）；**冷氣/洗衣機**客單較低（$1,299–$2,999），CPA 抓緊（例如 NT$150–350/表單）。
  - 轉換率參考：到達頁 3–8%（漏水急需型通常較高）。
  - 首月重點不是 ROAS，而是**累積足夠轉換讓智慧出價能學**（每活動每週 ≥ 15–30 轉換為佳；達不到就先用人工 CPC）。

**原則**：一條服務線一個活動、一個地區一個廣告群組（或用地區關鍵字 + 地區出價調整），避免一開始就切太碎導致每格數據太少。

## 4. 轉換追蹤設定（第 0 階段，最優先）

> 沒有這段，其他都別做。順序如下：

1. **建立/確認 GA4 資源**，取得真實 `G-XXXXXXXXXX` 評估 ID。→ **業主**（Google 帳號）＋ Claude 指導。
2. **把真實 ID 填進 `header.js` 的 `GA4_ID` 常數**（3 處佔位一次到位）。→ **Claude 或 Codex**（一行常數）。
3. **在 GA4 把 `quote_submit` 標為「關鍵事件」（主轉換）、`line_click` 標為次轉換。** → **業主**在 GA4 介面點選（Claude 給步驟）。
4. **連結 GA4 ↔ Google Ads**，把上述關鍵事件**匯入為 Google Ads 轉換**（或改用 Google Ads gtag 直接在 `quote_submit` 觸發轉換）。→ **業主**授權 + **Claude** 設定規格。
5. （進階）**強化轉換 Enhanced Conversions**：表單已收電話，可雜湊後回傳提升歸因準確度。→ Claude 評估，Codex 實作前端。
6. **UTM 規範**：所有廣告最終網址帶 `utm_source=google&utm_medium=cpc&utm_campaign={活動}&utm_content={廣告}&utm_term={關鍵字}`。→ Claude 定規範。
   - ⚠️ 注意：客戶按 LINE 深連結後 UTM 不一定帶進 bot，但**轉換已在網站端計數**，不影響 Google Ads 優化；bot 的 `utm_json` 只作次要來源回溯。

**驗收**：用 GA4 即時報表 + Google Ads 轉換診斷，實際送一次假表單，確認轉換有記到。

## 6. 廣告文案（Claude 產出，業主終審）

- **回應式搜尋廣告（RSA）**：每廣告群組 15 標題 × 4 說明，涵蓋：服務名＋地區、透明報價/免費初判、專業到府、服務範圍、信任訊號、行動呼籲。
- **範例（漏水抓漏，最終以完整清單為準）**：
  - 標題：「科技抓漏 免費初判」「{地區}漏水抓漏 專業到府」「壁癌漏水 現場透明報價」「熱像儀精準定位」「不施工不收費」
  - 說明：「紅外線熱像儀＋管道內視鏡精準找漏，現場報價不亂喊。雙北基桃竹苗中到府，加 LINE 快速預約。」
- **範例（冷氣清洗）**：
  - 標題：「分離式冷氣清洗 $1,499 起」「冷氣清洗 專業到府」「{地區}洗冷氣推薦」
  - 說明：「深層清洗除霉除臭，價格透明。加 LINE 預約，快速安排師傅到府。」
- **廣告額外資訊（Claude 備稿）**：網站連結（各服務頁）、標註（免費初判/透明報價/服務七縣市/加 LINE 預約）、結構化摘要（服務項目）、**LINE/通話**（本站以 LINE 為主，不放公開電話 CTA，符合現行策略）。
- ⚠️ **政策/商標**：**不得**在關鍵字或文案使用「洗洋洋」等競品商標（僅內部作價格對標，不對外）；避免最高級/療效式不實宣稱；漏水「保固」用語需與實際條件一致。

## 7. 到達頁策略（Codex 主導視覺與前端）

- **首月**可先用現有服務頁（`leak-repair.html` / `aircon.html`）當到達頁，但轉換率有限。
- **建議做「廣告專用轉換型 LP」**（漏水、冷氣各一先行）：單一強 CTA（加 LINE / 開表單）、首屏信任證明（案例照、儀器、服務區域、價格錨點）、精簡 FAQ、手機優先、秒開、對齊品牌質感（見 `FRONTEND-CRAFT.md`）。
- LP 必須埋 `quote_open / quote_submit / line_click` 事件（沿用 `header.js` 機制）。
- 廣告的「最終到達網址」對應服務；地區廣告可帶地區頁或 LP 的地區變體。

## 8. 出價、預算、地區、時段

- **出價**：轉換 < 15/週 前用**人工 CPC 或最大化點擊**控節奏；累積後切**最大化轉換 → 目標 CPA**。
- **預算（建議起步，最終由業主定）**：每條主活動 **NT$300–600/日**；首月總量約 **NT$15,000–30,000** 試水溫。漏水線可分較多預算（客單高）。
- **地區鎖定**：只投七個服務縣市（用「實際位於」而非「感興趣」）；雙北可加碼出價。
- **時段/裝置**：手機為主（家庭客群）；漏水急需型可全天，清潔類可加重白天與週末；依數據調整。

當進入 Performance Max / Demand Gen / 多媒體再行銷時需要：

- Logo（透明背景，橫式與方形）、品牌色系一致。
- 真實案例/儀器/施工情境照（去背或情境合成），對齊 `FRONTEND-CRAFT.md` 的質感標準。
- 多尺寸回應式多媒體廣告 banner。
- LP 的視覺與互動（果凍感/微互動），避免「呆版 AI 感」。
